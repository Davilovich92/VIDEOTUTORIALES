Atributos style

El atributo style sirve para aplicarle formato a través del lenguaje CSS

Texto en negrita. No debería usarse, porque lo correcto es mediante CSS

Este texto es importante, se ve igual que la negrita, pero es más correcto. Realmente tiene un sentido semántico de indicar que algo es importante

La equiqueta i de itálica (cursiva) tampoco debe utilizarse. Se debe aplicar con CSS.

Este tipo de texto enfatizado es parecido a la cursiva pero con un significado de énfasis, más correcto
